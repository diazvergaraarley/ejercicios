# GAG - Generic Adventure Game

## Description

A console-based RPG game inspired by MMORPG mechanics. Players can create a character, explore zones, fight enemies, complete missions, and progress through levels.

## Features (Planned)

* Character creation
* Combat system
* Inventory system
* Missions and quests
* Multiple zones
* Shop system

## Project Structure

Each system is modularized into separate folders:

* player/
* combat/
* inventory/
* world/
* missions/
* shop/

## Team Workflow

We are using Git branches for each feature:

* feature/player-system
* feature/combat-system
* etc.

All changes are merged into develop before going to main.

## How to Run

bash
python main.py
